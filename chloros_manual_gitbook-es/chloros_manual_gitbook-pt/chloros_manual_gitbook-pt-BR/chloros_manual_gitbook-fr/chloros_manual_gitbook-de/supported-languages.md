## Πώς να αλλάξετε γλώσσα

### Στο Chloros Desktop/Browser

1. Ανοίξτε τις ρυθμίσεις της εφαρμογής
2. Μεταβείτε στο μενού επιλογής γλώσσας
3. Επιλέξτε τη γλώσσα που προτιμάτε από τη λίστα
4. Η διεπαφή θα ενημερωθεί αμέσως

### Στο Chloros CLI

Χρησιμοποιήστε την παράμετρο γλώσσας με τις εντολές σας για να ορίσετε τη γλώσσα διεπαφής.

### Στο Chloros Python SDK

Ρυθμίστε την παράμετρο γλώσσας κατά την προετοιμασία του SDK για να λαμβάνετε μηνύματα και εξόδους στη γλώσσα που προτιμάτε.

## Κάλυψη

Και οι 38 γλώσσες υποστηρίζονται πλήρως σε:

* **Chloros Desktop** - Ολοκληρωμένη μετάφραση GUI
* **Chloros Browser** - Διασύνδεση Ιστού σε όλες τις γλώσσες
* **Chloros CLI** - Διεπαφή γραμμής εντολών και μηνύματα εξόδου
* **Chloros Python SDK** - Μηνύματα και τεκμηρίωση API

Η υποστήριξη γλώσσας διασφαλίζει ότι οι χρήστες σε όλο τον κόσμο μπορούν να εργάζονται αποτελεσματικά στη μητρική τους γλώσσα χωρίς εμπόδια.
