# Μορφές εξόδου εικόνας

### TIFF (16-bit)

16-bit (ψηφιακός αριθμός pixel 0 - 65535) Έξοδος μορφής TIFF (.tif).

### TIFF (32-bit, Ποσοστό %)

32-bit (πλωτή τιμή pixel 0,0 - 1,0) Έξοδος μορφής TIFF (.tif). Οι τιμές pixel αντιπροσωπεύουν ποσοστά ανακλαστικότητας όπου 0,0 = 0% και 1,0 = 100%.

### **JPG (8-bit)**

8-bit (ψηφιακός αριθμός pixel 0 -255) Έξοδος σε μορφή JPG (.jpg).

### **PNG (8-bit)**

Έξοδος σε μορφή PNG (.png) 8-bit (ψηφιακός αριθμός pixel 0 - 255). Συμπίεση χωρίς απώλειες, κατάλληλη για προβολή και οπτικοποίηση στο διαδίκτυο.
