## 📝 Επισκόπηση σεναρίων

### Κύρια καθημερινή ροή εργασίας
**`update_all_translations.py`**
- Ενημερώνει και τα 37 αποθετήρια γλωσσών
- Συγχρονίζει κείμενο, εικόνες και στοιχεία
- Μεταφράζει μόνο τα αρχεία που έχουν αλλάξει
- Αυτόματη υποβολή και αποστολή στο GitHub
- Χρήση: `python update_all_translations.py`

### Σενάρια μετάφρασης
**`translate_with_deepl.py`**
- Βασική μετάφραση DeepL (32 γλώσσες)
- Διαχειρίζεται περιγραφές frontmatter
- Πλήρης προστασία markdown

**`translate_with_google.py`**
- Ενσωμάτωση Google Translate (5 γλώσσες)
- Ίδια προστασία με το DeepL
- Χειρίζεται τους περιορισμούς του API

**`translate_google_conservative.py`**
- Εξαιρετικά αργό αλλά αξιόπιστο Google Translate
- Μετάφραση γραμμή προς γραμμή
- Μεγάλες καθυστερήσεις για την αποφυγή περιορισμών ταχύτητας
- Για δύσκολες γλώσσες: `python translate_google_conservative.py hi`

### Σενάρια χρησιμότητας
**`verify_all_pushed.py`**
- Έλεγχος ότι και τα 37 αποθετήρια έχουν μεταφερθεί στο GitHub

**`check_google_progress.py`**
- Έλεγχος του αριθμού των αρχείων γλώσσας του Google Translate

**`check_hindi_progress.py`**
- Λεπτομερής πρόοδος της μετάφρασης στα Χίντι

**`push_until_stable.py`**
- Μεταφέρετε όλα τα αποθετήρια μέχρι να μην υπάρχουν αλλαγές
