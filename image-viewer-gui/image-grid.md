# Πλέγμα εικόνων

Αφού εισαγάγετε εικόνες σε ένα έργο, θα δείτε τις εικόνες να είναι διατεταγμένες σε ένα πλέγμα στην κύρια περιοχή.

## Μέγεθος μικρογραφιών

Μπορείτε να χρησιμοποιήσετε το ρυθμιστικό ζουμ στην επάνω δεξιά γωνία για να προσαρμόσετε το μέγεθος των μικρογραφιών των εικόνων. Μπορείτε επίσης να κρατήσετε πατημένο το CTRL + τροχό του ποντικιού για να τις κλιμακώσετε.

## Κουμπιά προβολής εξαγωγής

Καθώς οι εικόνες υποβάλλονται σε επεξεργασία, θα εμφανιστούν κουμπιά στην επάνω αριστερή γωνία της περιοχής του πλέγματος. Αυτά τα κουμπιά αλλάζουν τις μικρογραφίες του πλέγματος μεταξύ διαφορετικών τύπων εικόνων:

* **JPG** - Αρχικές μικρογραφίες JPG της κάμερας
* **Στόχοι** - Εικόνες που περιέχουν ανιχνευμένους στόχους βαθμονόμησης
* **Αντανάκλαση** - Βαθμονομημένες εικόνες αντανάκλασης
* **Δείκτης/LUT** - Υπολογισμένες εικόνες πολυφασματικού δείκτη με χρωματική διαβάθμιση LUT

Κάνοντας κλικ στις μικρογραφίες του πλέγματος εικόνων ανοίγει η πλήρης οθόνη [Image Viewer](opening-an-image-full-screen.md). Εάν η λειτουργία προβολής στο πλέγμα έχει ρυθμιστεί σε μια συγκεκριμένη εξαγωγή όπως &quot;Στόχοι&quot;, όταν κάνετε κλικ στη μικρογραφία του πλέγματος θα ανοίξει η εξαγόμενη εικόνα στόχου σε πλήρη οθόνη.
